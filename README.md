# da-sprint — 数据分析面试冲刺教练（Claude Skill）

> **代码交给 AI，思考必须是你自己的。**
> da-sprint 不教你背语法。它带你完整经历一次真实的数据分析:
> 真实业务问题 → 自己提出假设 → 挖掘规律 → 假设检验与控混杂 → 产出有数据支撑的结论与改进方向。
> 终点是一个从假设到结论都出自你自己思考、经得起面试追问的项目。

## 给谁用

- **运营 / 行业研究 / 风控 / 产品**岗求职者：JD 里写着"数据分析能力"，你有分析逻辑，但写不出代码
- 经济学 / 社科背景转行者：会 Stata / SQL / Excel？教练会自动切换**对照翻译教学**（如 `merge 1:1` ↔ `validate='1:1'`）
- ❌ 不适合：想系统学 Python 工程、或算法岗刷题的人

## 它和看课、问 ChatGPT 有什么不同

**1. 假设驱动，项目是你自己的调查。** 开场你自己提出"哪里可能出问题"的假设清单，
全程逐条验证**你的**假设。没有照抄的标准项目——面试官问"为什么查这个"，答案天然是你的。

**2. 代码交给 AI，决策必须归你。** 课程验收的不是语法默写，是业务岗真正被考的三种能力：
**读码**（看懂 AI 写了什么）、**验收**（AI 输出错了你能发现）、**指挥**（下达精确到不会做错的指令）。
但每个分析决策——缺失怎么处理、分箱边界、指标口径——必须你亲自做并说出理由,
含糊的理由会在模拟面试中被原样打回。

**3. 真数据、真洞察、有方法论底子。** 数据是巴西电商 Olist 的 2 万条真实订单（自带真实的脏）。
配送时长与差评率的 8 倍梯度、"违约比绝对慢更致命"的中介机制，都是真实商业规律，
面试官可查证。分析框架内置**变量构造五步 SOP** 和**机制证据五件套**
（Bradford Hill 准则的业务版：双环验证 / 剂量反应 / 分层稳健 / 排除替代解释 / 声明证据边界），
学员的每个结论都标注证据等级。

## 你会经历什么

四个阶段，自定步速（连夜刷完或拉一周均可），阶段之间有验收门禁：

| 阶段 | 内容 | 产出 |
|---|---|---|
| 一 清洗 | 开箱五眼、缺失机制、去重、类型 | 数据质量报告 + 清洗脚本 |
| 二 假设初验 | 你的假设逐条上交叉分析 | 趋势图 + 初步发现 |
| 三 核心洞察 | 分层防混杂、logistic 回归稳健性、中介机制 | 项目核心结论 |
| 四 成稿面试 | STAR 讲述稿、AI 面试官 10 题追问 | 完整交付物 + 薄弱点清单 |

## 你会带走什么

- `analysis.py`：可复跑的完整分析流水线（清洗 → 假设验证 → 回归 → 反事实测算）；项目是否公开上传 GitHub 由你自己决定——它的价值在于每个决策是你做的，不在于和别人雷同的展示
- `项目讲述稿.md`：一句话电梯陈述 + STAR + **必背数字卡** + 预测追问与标准回应 + 完整回归表附录
- `learner_log.md`：你的全部决策记录与薄弱点清单（也是断点续学的存档）

## 快速开始

**Claude Code（推荐，token 消耗低一个量级）**
```bash
# 解压 da-sprint.skill(即 zip)到项目的 .claude/skills/ 目录
unzip da-sprint.skill -d .claude/skills/
# 然后对 Claude 说:"教我数据分析" 或 "继续上课"(带 learner_log.md 可断点续学)
```

**claude.ai**：上传 `.skill` 文件启用,直接开聊。

## 真实陪跑记录

本产品由首位真实学员（经济学背景、pandas 零词汇）陪跑打磨：单日从"我不懂要填啥"
推进到 logistic 回归与中介机制发现，过程中触发产品迭代 9 次（词汇填空废除、
阶段制取代天数制、假设驱动重构、人设分流验收……），全部教训已写入 SKILL.md 教学铁律。

## 结构

```
da-sprint/
├── SKILL.md                 # 教练人格与教学铁律
├── scripts/prepare_data.py  # 数据准备(参数化再脏化,每个学员数据不同)
├── data/                    # Olist 2万订单子集(7张表)
└── references/
    ├── curriculum.md        # 四阶段12卡剧本(含教练埋点注记)
    ├── idioms.md            # 25句型表 + Stata→pandas 对照
    ├── sop.md               # 变量构造五步SOP + 统计问答
    ├── interviewer.md       # 模拟面试官指南
    └── backgrounds.md       # 巴西电商业务背景(防归因翻车)
```
